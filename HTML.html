HTML คืออะไร?

HTML (HyperText Markup Language) เป็นภาษาเครื่องหมายที่ใช้ในการสร้างและจัดรูปแบบเอกสารบนเว็บ HTML เป็นพื้นฐานของเว็บเพจทุกชนิดและใช้ในการสร้างโครงสร้างของเว็บ เช่น หัวเรื่อง, พารากราฟ, ลิงค์, และรูปภาพ

จุดเด่นของ HTML

HTML มีจุดเด่นในการสร้างเอกสารที่มีโครงสร้างชัดเจนและสามารถจัดรูปแบบข้อมูลต่างๆ บนเว็บได้อย่างง่ายดาย นอกจากนี้ HTML ยังสามารถใช้ร่วมกับ CSS และ JavaScript เพื่อเพิ่มการจัดรูปแบบและฟังก์ชันการทำงานให้กับเว็บเพจได้

โครงสร้างพื้นฐานของเอกสาร HTML

เอกสาร HTML ประกอบด้วยส่วนหลักๆ ดังนี้:

<!DOCTYPE html>
<html lang="th">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>หัวข้อของเอกสาร</title>
    <style>
        /* การกำหนดสไตล์ */
    </style>
</head>
<body>
    <header>
        <h1>หัวเรื่องหลัก</h1>
    </header>
    <main>
        <section>
            <h2>หัวเรื่องรอง</h2>
            <p>เนื้อหาของพารากราฟ</p>
        </section>
    </main>
    <footer>
        <p>ข้อมูลติดต่อ</p>
    </footer>
</body>
</html>

1. การประกาศ <!DOCTYPE html>

คำสั่งนี้ใช้เพื่อบอกเบราว์เซอร์ว่าเอกสารนี้เป็น HTML5:

<!DOCTYPE html>

2. การใช้ <html> แท็ก

แท็กนี้ใช้เป็นแท็กหลักของเอกสาร HTML และมีแอตทริบิวต์ lang เพื่อระบุภาษาของเอกสาร:

<html lang="th">
    ...
</html>

3. การใช้ <head> และ <title>

แท็ก <head> ใช้สำหรับข้อมูลเมตาของเอกสาร เช่น ชื่อเรื่อง, การเชื่อมโยงกับไฟล์ CSS, และการกำหนดลักษณะการแสดงผลบนอุปกรณ์มือถือ:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ชื่อของเอกสาร</title>
</head>

4. การใช้ <body>

แท็ก <body> ใช้สำหรับเนื้อหาหลักของเอกสาร เช่น หัวเรื่อง, พารากราฟ, และลิงค์:

<body>
    <h1>หัวเรื่องหลัก</h1>
    <p>นี่คือลิงค์ไปยัง <a href="https://www.example.com">เว็บไซต์</a> ของเรา</p>
</body>

5. การใช้ <header>, <main>, และ <footer>

แท็ก <header> ใช้สำหรับส่วนหัวของเอกสาร, <main> ใช้สำหรับเนื้อหาหลัก, และ <footer> ใช้สำหรับข้อมูลติดต่อหรือข้อเสนอแนะ:

<header>
    <h1>หัวเรื่องหลัก</h1>
</header>

<main>
    <section>
        <h2>หัวเรื่องรอง</h2>
        <p>เนื้อหาของพารากราฟ</p>
    </section>
</main>

<footer>
    <p>ข้อมูลติดต่อ</p>
</footer>

6. การใช้ <a> แท็ก

แท็ก <a> ใช้สำหรับสร้างลิงค์ไปยังหน้าเว็บหรือเอกสารอื่น:

<a href="https://www.example.com">ไปยังเว็บไซต์</a>

7. การใช้ <img> แท็ก

แท็ก <img> ใช้สำหรับแทรกรูปภาพในเอกสาร HTML:

<img src="image.jpg" alt="คำบรรยายรูปภาพ">

8. การใช้ <form> แท็ก

แท็ก <form> ใช้สำหรับสร้างแบบฟอร์มที่ให้ผู้ใช้กรอกข้อมูล:

<form action="/submit" method="post">
    <label for="name">ชื่อ:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="ส่ง">
</form>

9. การใช้ <table> แท็ก

แท็ก <table> ใช้สำหรับสร้างตารางในเอกสาร HTML:

<table>
    <thead>
        <tr>
            <th>หัวข้อ 1</th>
            <th>หัวข้อ 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>ข้อมูล 1</td>
            <td>ข้อมูล 2</td>
        </tr>
    </tbody>
</table>
