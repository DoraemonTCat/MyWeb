Python คืออะไร?

Python เป็นภาษาโปรแกรมที่ได้รับความนิยมสูงและถูกพัฒนาโดย Guido van Rossum ภาษา Python ถูกออกแบบมาให้มีไวยากรณ์ที่อ่านง่ายและสะดวกสำหรับการเขียนโปรแกรม สามารถใช้งานได้ในหลากหลายประเภทของโปรเจ็กต์ เช่น การพัฒนาเว็บ, การวิเคราะห์ข้อมูล, และการพัฒนาแอปพลิเคชัน

จุดเด่นของ Python

Python มีจุดเด่นหลายประการ เช่น ความง่ายในการอ่านและเขียนโค้ด, การสนับสนุนการพัฒนาซอฟต์แวร์ได้หลากหลายประเภท, และมีไลบรารีและเฟรมเวิร์กที่ครอบคลุมสำหรับการทำงานต่างๆ รวมถึงการพัฒนา Web Application และ Data Analysis

คำสั่งพื้นฐานใน Python

1. การใช้คำสั่ง if และ else

คำสั่ง if และ else ใช้ในการตัดสินใจว่าจะแสดงผลลัพธ์ใดขึ้นอยู่กับเงื่อนไขที่กำหนด:

number = 10

if number > 0:
    print("Number is positive.")
elif number < 0:
    print("Number is negative.")
else:
    print("Number is zero.")

ในตัวอย่างนี้ โปรแกรมจะตรวจสอบค่าของตัวแปร number และพิมพ์ข้อความที่เหมาะสมออกมา

2. การใช้คำสั่ง for

คำสั่ง for ใช้ในการทำงานวนลูปโดยรู้จำนวนรอบที่ต้องการ:

for i in range(1, 6):
    print("Iteration", i)

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5

3. การใช้คำสั่ง while

คำสั่ง while ใช้ในการทำงานวนลูปตราบใดที่เงื่อนไขยังเป็นจริง:

i = 1
while i <= 5:
    print("Iteration", i)
    i += 1

ในตัวอย่างนี้ โปรแกรมจะทำการพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5 เช่นเดียวกับคำสั่ง for

4. การใช้คำสั่ง switch (โดยใช้ match ใน Python 3.10 ขึ้นไป)

ใน Python รุ่นก่อนหน้าไม่มีคำสั่ง switch แต่ใน Python 3.10 ขึ้นไปมีการเพิ่มคำสั่ง match ซึ่งทำหน้าที่คล้ายกัน:

day = 3

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case 4:
        print("Thursday")
    case 5:
        print("Friday")
    case 6:
        print("Saturday")
    case 7:
        print("Sunday")
    case _:
        print("Invalid day")

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ชื่อของวันในสัปดาห์ตามค่าของตัวแปร day โดยใช้คำสั่ง match
