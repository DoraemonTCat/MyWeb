C++ คืออะไร?

C++ เป็นภาษาการเขียนโปรแกรมที่พัฒนาขึ้นจากภาษา C โดยเพิ่มคุณสมบัติของการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming) เข้ามา C++ ใช้สำหรับการพัฒนาแอปพลิเคชันต่างๆ รวมถึงระบบปฏิบัติการ เกม และซอฟต์แวร์ประสิทธิภาพสูง

จุดเด่นของ C++

C++ มีจุดเด่นในการรองรับการเขียนโปรแกรมเชิงวัตถุ, ความสามารถในการจัดการหน่วยความจำ, และประสิทธิภาพที่สูง นอกจากนี้ยังสามารถใช้ในการพัฒนาแอปพลิเคชันที่ต้องการประสิทธิภาพและการควบคุมที่แม่นยำ

คำสั่งพื้นฐานใน C++

1. การใช้คำสั่ง if และ else

คำสั่ง if และ else ใช้ในการตัดสินใจว่าจะทำการดำเนินการใดขึ้นอยู่กับเงื่อนไขที่กำหนด:

#include <iostream>
using namespace std;

int main() {
    int number = 10;

    if (number > 0) {
        cout << "Number is positive." << endl;
    } else if (number < 0) {
        cout << "Number is negative." << endl;
    } else {
        cout << "Number is zero." << endl;
    }

    return 0;
}

ในตัวอย่างนี้ โปรแกรมจะตรวจสอบค่าของตัวแปร number และพิมพ์ข้อความที่เหมาะสมออกมา

2. การใช้คำสั่ง for

คำสั่ง for ใช้ในการทำงานวนลูปโดยรู้จำนวนรอบที่ต้องการ:

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iteration " << i << endl;
    }

    return 0;
}

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5

3. การใช้คำสั่ง while

คำสั่ง while ใช้ในการทำงานวนลูปตราบใดที่เงื่อนไขยังเป็นจริง:

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 5) {
        cout << "Iteration " << i << endl;
        i++;
    }

    return 0;
}

ในตัวอย่างนี้ โปรแกรมจะทำการพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5 เช่นเดียวกับคำสั่ง for

4. การใช้คำสั่ง switch

คำสั่ง switch ใช้ในการตรวจสอบค่าของตัวแปรและทำการดำเนินการตามกรณีที่กำหนด:

#include <iostream>
using namespace std;

int main() {
    int day = 3;

    switch (day) {
        case 1:
            cout << "Monday" << endl;
            break;
        case 2:
            cout << "Tuesday" << endl;
            break;
        case 3:
            cout << "Wednesday" << endl;
            break;
        case 4:
            cout << "Thursday" << endl;
            break;
        case 5:
            cout << "Friday" << endl;
            break;
        case 6:
            cout << "Saturday" << endl;
            break;
        case 7:
            cout << "Sunday" << endl;
            break;
        default:
            cout << "Invalid day" << endl;
    }

    return 0;
}

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ชื่อของวันในสัปดาห์ตามค่าของตัวแปร day โดยใช้คำสั่ง switch
