Java คืออะไร?

Java เป็นภาษาโปรแกรมที่ถูกพัฒนาโดย Sun Microsystems ในปี 1995 Java ถูกออกแบบมาเพื่อให้สามารถทำงานได้บนทุกแพลตฟอร์มที่สนับสนุน Java โดยไม่ต้องมีการคอมไพล์ใหม่สำหรับแต่ละแพลตฟอร์ม

คำสั่งพื้นฐานใน Java

1. การใช้คำสั่ง if และ else

คำสั่ง if และ else ใช้ในการตัดสินใจว่าจะแสดงผลลัพธ์ใดขึ้นอยู่กับเงื่อนไขที่กำหนด:

int number = 10;

if (number > 0) {
    System.out.println("Number is positive.");
} else if (number < 0) {
    System.out.println("Number is negative.");
} else {
    System.out.println("Number is zero.");
}

ในตัวอย่างนี้ โปรแกรมจะตรวจสอบค่าของตัวแปร number และพิมพ์ข้อความที่เหมาะสมออกมา

2. การใช้คำสั่ง for

คำสั่ง for ใช้ในการทำงานวนลูปโดยรู้จำนวนรอบที่ต้องการ:

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration " + i);
}

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5

3. การใช้คำสั่ง while

คำสั่ง while ใช้ในการทำงานวนลูปตราบใดที่เงื่อนไขยังเป็นจริง:

int i = 1;
while (i <= 5) {
    System.out.println("Iteration " + i);
    i++;
}

ในตัวอย่างนี้ โปรแกรมจะทำการพิมพ์ข้อความ "Iteration" ตามด้วยหมายเลขลูปตั้งแต่ 1 ถึง 5 เช่นเดียวกับคำสั่ง for

4. การใช้คำสั่ง switch

คำสั่ง switch ใช้ในการเลือกทำงานจากหลายตัวเลือกตามค่าของตัวแปร:

int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    case 4:
        System.out.println("Thursday");
        break;
    case 5:
        System.out.println("Friday");
        break;
    case 6:
        System.out.println("Saturday");
        break;
    case 7:
        System.out.println("Sunday");
        break;
    default:
        System.out.println("Invalid day");
        break;
}

ในตัวอย่างนี้ โปรแกรมจะพิมพ์ชื่อของวันในสัปดาห์ตามค่าของตัวแปร day
